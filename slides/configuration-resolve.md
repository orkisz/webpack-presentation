##  Configuration - Resolve

Controls how files are resolved with *require* method

```
resolve: {
      extensions: Array<string>, // ['', '.ts', '.js', '.json'],
      root: string, // usually absolute path to './src'
      modulesDirectories: Array<string> // ['node_modules']
}
```
